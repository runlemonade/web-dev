Development Timeline

Our roadmap to building the complete digital fabrication ecosystem

July 1st, 2025
Publishing of FORM
The official launch of FORM for SketchUp, bringing professional cabinet modeling to the masses.

Goals for July

  • Add 10 new models to FORM library
  • Publish to SketchUp Extension Warehouse
  • Establish user community and feedback channels
August 31st, 2025
DRAFT & NESTMATE FREE Release
Launching the free versions of DRAFT and NESTMATE, providing essential tools for 2D layout and basic nesting capabilities.

DRAFT FREE Features

  • 2D layout and vector design tools
  • Basic drawing and measurement tools
  • Export to SVG/DXF

NESTMATE FREE Features

  • CSV panel saw nesting
  • SketchUp nesting integration
  • Basic material optimization

FORM Updates

  • Continued additional models
  • Enhanced user experience
  • Performance improvements
October 31st, 2025
NESTMATE PRO Release
The professional nesting solution with advanced production features and CNC integration.

Production Features

  • Advanced CNC nesting algorithms
  • Auto cutting and extruding tools
  • CNC production optimization
  • Export to DXF with production data
  • Continued support for NESTMATE FREE
