FAQ's

How do I start my free trial?

Simply download the FORM extension (.rb file) and install it in SketchUp through the Extensions menu. You'll have immediate access to the full library for 7 days with no credit card required.

What's included in FORM Professional?

You get access to 215+ modular cabinet models with layout files, full technical blueprints for every model, beta access to NestMate nesting extension, Shaker and in-frame units (coming July), monthly content drops with new modules and layout packs, advanced search and filtering, and priority support.

Do I need SketchUp Pro to use FORM?

Yes, a copy of SketchUp Pro is required to use features such as dynamic components and LayOut integration that FORM relies on.

Can I use these components for commercial projects?

Yes — FORM Professional includes a full commercial use license. You can use all components in client work, commercial projects, or any professional applications without restrictions.

What happens after my 7-day trial ends?

After 7 days, you'll need to subscribe for £49.99/month to continue using FORM. If you don't subscribe, the extension will stop working, but any models you've already inserted into your SketchUp files will remain.
